Розділ 1: HTML та CSS

HTML (HyperText Markup Language) — це мова розмітки, яка використовується для створення структури веб-сторінок. CSS (Cascading Style Sheets) дозволяє оформляти ці сторінки, задаючи стиль елементів.

Примітка: HTML описує "що", а CSS — "як".

Розділ 2: JavaScript

JavaScript — це мова програмування, яка додає інтерактивність на веб-сторінки. Вона дозволяє створювати динамічний контент, наприклад, анімації, форми, що реагують на події, та багато іншого.

Примітка: JavaScript тісно пов'язаний із HTML і CSS, утворюючи "основу веб-розробки".

Розділ 3: Веб-дизайн

Гарний веб-дизайн поєднує естетику та функціональність. Він має бути зрозумілим для користувача, зручним у використанні та адаптивним для різних пристроїв.

Примітка: Адаптивний дизайн дозволяє веб-сайту добре виглядати на будь-якому пристрої.
